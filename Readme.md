## Todo List Use Phyton

This is a simple CRUD app using Phyton

How to run this project?

```bash
docker-compose up
```

How to stop this project?

```bash
docker-compose down
```

How to run test?

```bash
pytest --cov=app --cov-report=html
```

How to see coverage?

```bash
open ./htmlcov/index.html
```

try it!
